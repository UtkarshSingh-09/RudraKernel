# SIEGE Project Readme (Docs Mirror)

This document mirrors the root README structure used for hackathon submission.

## 1) Problem
SIEGE trains agents to reason under deceptive multi-agent incident response.

## 2) Environment
Agents receive live incident observations and can call diagnose/challenge/ratify/escalate/whisper/postmortem.

## 3) Results
See plots in docs/plots and W&B links in root README.

## 4) Why It Matters
Calibrated, adversarially-robust reasoning improves reliability in safety-critical operations.
